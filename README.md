### 🏥 [Well Appointment](https://wellappointment.com)
> Production-ready healthcare SaaS for doctors and patients

- Unique **Care ID system** — each doctor gets a personal URL + QR code for instant booking
- Full **Doctor Dashboard** — schedule management, appointment requests, real-time notifications
- Full **Patient Dashboard** — booking, tracking, email confirmations
- **Video Consultation** — built-in telehealth for remote doctor-patient meetings
- **Real-time data sync** — high-concurrency scheduling across both sides
- **Live pricing system** — Lifetime and Pro/Monthly plans
- Built solo in 5 months · Next.js · Node.js · MongoDB · GCP
